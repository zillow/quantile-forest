Getting Started
---------------

Prerequisites
~~~~~~~~~~~~~

The quantile-forest package requires the following dependencies:

* python (>=3.8)
* numpy (>=1.23)
* scikit-learn (>=1.0)
* scipy (>=1.4)

Installation
~~~~~~~~~~~~

quantile-forest can be installed using `pip`::

  pip install quantile-forest
