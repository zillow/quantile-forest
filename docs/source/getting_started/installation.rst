Getting Started
---------------

Prerequisites
~~~~~~~~~~~~~

The quantile-forest package requires the following dependencies:

* python (>=3.9)
* numpy (>=1.23)
* scikit-learn (>=1.5)
* scipy (>=1.4)

Installation
~~~~~~~~~~~~

quantile-forest can be installed using `pip` or `conda`.

Using `pip`::

  pip install quantile-forest

Using `conda`::

  conda install quantile-forest -c conda-forge
